Arguments : '$event'
window:scroll(event)

HostListener Decorator a DOM event to listen for, and provides a handler method to run when that event occurs. Here attaches listener to window on scroll event. In onScroll method assigns headerClass property with the class "'fixed-top' | 'fixed-top expand'" based on window pageYOffset and set the tab active.
