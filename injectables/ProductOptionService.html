Public getProductOptions
getProductOptions(productId: string | Array, groupName: string, pricing: boolean)

Method gets a list of product option components for a given product

Example:

      import { ProductOptionService, ProductOptionComponent } from '@apttus/ecommerce';
      import { Observable } from 'rxjs/Observable';

      export class MyComponent implements OnInit{
           optionComponentList: Array<ProductOptionComponent>;
           optionComponentList$: Observable<Array<ProductOptionComponent>>;

           constructor(private productOptionService: ProductOptionService){}

           getProductOptions(productCode: string){
                this.productOptionService.getProductOptions(productCode).subscribe(o => this.optionComponentList = o);
                // or
                this.optionComponentList$ = this.productOptionService.getProductOptions(productCode);
           }
      }
Parameters :
Name Type Optional Default value Description
productId string | Array<string> No

the product id of the product with which to get the options for
groupName string No null

the option group to filter the results by
pricing boolean No false

boolean option to return pricing with the options (performs an extra callout, so may be slightly slower).
Returns : Observable<Array<ProductOptionComponent>>

an observable array of product options components for the given product.
